# Archived Laptop Roles

The Arch Linux laptop playbooks and supporting roles were retired from the maintained workflow and moved into the archive.

## Historical Stages

- `laptop_arch_install_1.yml` — initial partitioning, encryption, filesystems, and `pacstrap`
- `laptop_arch_install_2.yml` — hostname, users, bootloader, networking, and first boot configuration
- `laptop_config.yml` — post-install desktop and user-environment configuration

For the archived playbooks, see [`archive/playbooks/`](../../playbooks/). For the full archived role list, see [legacy-roles.md](legacy-roles.md).
